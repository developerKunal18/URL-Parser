# Day 196 – URL Query Parameter Parser

A Python script that extracts query parameters from a URL.

## Description
This project parses a URL and displays all query parameters in key-value format.

## Features
- URL parsing
- Query extraction
- Useful for APIs and debugging

## Technologies Used
- Python
- urllib.parse module

## How to Run
1. Clone the repository
2. Navigate to the project folder
3. Run:

   python query_parser.py

4. Enter URL

## Example

Input:
https://example.com/search?q=python&page=2

Output:
q = python  
page = 2
